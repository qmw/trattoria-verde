# Trattoria Verde 🍝

Strona-wizytówka fikcyjnej włoskiej trattorii — projekt do **portfolio stron internetowych dla firm**.

Rustykalno-edytorski klimat: kremowy papier z teksturą ziarna, głęboka oliwkowa zieleń, terakota i złoto jako akcenty. Jedna responsywna strona (one-page) z animacjami wejścia i ujawnianiem sekcji przy przewijaniu.

## ✨ Sekcje

- **Hero** — pełnoekranowe zdjęcie z powolnym zoomem i animowanym wejściem treści
- **O nas** — historia rodzinnej trattorii
- **Menu** — 4 kategorie (Antipasti / Pasta / Forno a legna / Dolci) z opisami i cenami
- **Galeria** — siatka zdjęć z efektem powiększenia na hover
- **Rezerwacja** — formularz rezerwacji stolika
- **Kontakt** — godziny otwarcia, dane kontaktowe i mapa

## 🛠️ Stack

- [Astro 5](https://astro.build/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- Typografia: Fraunces · Mulish · Caveat (Google Fonts)

## 🚀 Uruchomienie

```bash
npm install
npm run dev      # serwer deweloperski → http://localhost:4321
npm run build    # build produkcyjny do ./dist
npm run preview  # podgląd builda
```

## ☁️ Hosting

Projekt statyczny — gotowy do darmowego hostingu na [Vercel](https://vercel.com/) (import repozytorium → framework wykrywany automatycznie jako Astro).
